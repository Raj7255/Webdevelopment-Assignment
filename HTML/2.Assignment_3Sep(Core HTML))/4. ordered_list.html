1. 'head' : Contains metadata about the document, such as title, charset, and links to external stylesheets and scripts.
  2. 'body' : Contains the main content of the document, including any HTML elements, text, and images.
  3. 'h1' : Defines a level 1 header, typically used for sailing a main title or subheading.
  4. 'p' : Defines a paragraph of text, often used for describing a concept or providing information.
  5. 'ul' : Defines a list of items, often used for providing a list of options or items for the user to choose from.
  6. 'title' : Defines the title of the HTML document, which appears in the browser's title bar and search engine.
  7. 'header' : A section for navigational and informational items, such as site logo, navigation menu, and legal information.
