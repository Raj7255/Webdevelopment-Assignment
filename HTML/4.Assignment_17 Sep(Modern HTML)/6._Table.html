HTML HTML5
HTML (Hypertext Markup Language) is the standard markup language used to create web pages HTML5 is the latest version of the language, which includes new features and improvements
HTML does not have support for video and audio. HTML5 supports both video and audio.
The HTML browser uses cache memory as temporary storage. HTML5 has the storage options like:application cache, SQL database, and web storage.
HTML Does not allow JavaScript to run in browser. HTML5 Allows JavaScript to run in browser. This is possible due to JS Web worker API in HTML5.
